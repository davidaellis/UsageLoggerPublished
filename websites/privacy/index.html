Usage Logger 2 - Privacy Policy

Who collects the data?

The application was primarily designed so that it would be useful for psychological research. However, usage data can be collected for any purpose. The usage data collected will only be accessible by others if the end user chooses to share the encrypted file and accompanying password.

What data is collected?

Usage Logger 2 collects three types of data. Data regarding what the person had used their phone for in the past (up to 5 days in the past). This will include when the phone screen was on and what apps were used. Other data is also captured by the app, but this resembles noise about the internal operations of the phone. Data regarding the context of which the file is used, this. It can collect what apps are installed, the permissions requested by these apps and the reaction of these apps. Finally, the continuous PDF file will include data about when the phone was "off" or "on". It can also include when the screen is on/off, when apps were used and the associated identity of those apps, when notifications were sent or deleted and which apps generated the apps, also if there were any changes in the apps which were installed on the device. What the app collects depends on the information stored in the QR-code provided to the user. Once the user has scanned in this QR code to participate in an experiment, they will be informed prior to begin of data collection what data will be collected from their device. .

How secure is the data?

The data is far more secure than the Standard Android security protocols. A security key is used to protect the data, this is generated through utilising user generated randomness. This key is then stored in an encrypted file, only accessible to the app. The files associated with the data are protected with a SHA 256 bit encryption. IMPORTANT – Android users with rooted phones or who have reason to believe that they have compromised security should not use this application. We will also notify the user of this prior to data collection. It is the users responsibility to ensure the integrity of their smartphone security before installing this application.

What is the legal basis for processing the data?

The data will only be accessible to researchers when a user sends the encrypted files and accompanying password. This is interpreted as giving consent to process the data for the purposes of scientific research. IMPORTANT FOR USERS AND RESEARCHERS – Files and passwords should not be sent in the same email.

Will the collected data be shared with the third parties?

The user has complete control over who has access to their data. App data is not stored online, so there is now way of accessing the data on a users phone. Users must export the data files and forward their password to the intended recipient in order to provide access to the data.

How will the data be used?

The data collected is intended to be used by scientists who are attempting to better understand how individual and group movements relate to other scientific phenomena. We strongly discourage anyone not engaging in scientific research from using the app.

How long will the data be stored?

The data will be stored for as long as the individual has the application installed on their phone. When the user decides to uninstall the application, all associated data will be deleted.The storage of the data by anyone that the participant has consciously shared with their data is not within our control and is not our responsibility.

What rights does the research participant have?

If used as part of scientific research, a participant should be informed regarding the nature of the research in line with standard ethical procedures. Similarly, participants should have been informed by researchers directly regarding how sent data will be processed and/or shared. The application will provide permanent written and icon based notifications to remind users that data is being collected. The application provides participants ability to access their own data. Participants can withdraw at any point by simply uninstalling the application, which also deletes all subsequent data.
