This website allows researchers to customise UsageLogger2. It is possible to specify the type or types of data to be collected. The order of collection can also be configured. Click a section headline below to reveal more information.

Select Data Sources

  • Contextual data provides a snapshot of what software is installed on the smartphone. This can include installed apps, requested permissions and/or permissions granted.
  • Continuous logging records smartphone interactions after installation. Features captured include screen on/off events, specific app usage and notifications received.
  • Past usage extracts data from an internal database of historical data maintained by Android.

Customise Order

Customise the order in which data is collected by dragging and dropping the text on the right hand of the screen. The text only appears after one or more data sources have been selected.

Generate a QR Code

Clicking the 'Create QR-Code' button finalizes the customisation. A QR-Code is then generated. This code must then be scanned with the UsageLogger2 app. It loads the configuration into the app and tells the app what data to collect. The QR code can also be downloaded for future use (configurations).
